# File Butler (文件管家)

AI की मदद से अपने अस्त-व्यस्त फ़ोल्डर व्यवस्थित करें और साफ़ करने लायक फ़ाइलें खोजें। **macOS और Windows** पर समर्थित, पूरी तरह ऑफ़लाइन चल सकता है।


---

## ✨ दो मुख्य सुविधाएँ

### 🗂️ फ़ाइल व्यवस्थापन
एक फ़ोल्डर चुनें, AI हर फ़ाइल को समझता है (सिर्फ़ फ़ाइल नाम नहीं देखता) और स्वचालित रूप से श्रेणियाँ तथा अधिक सार्थक फ़ाइल नाम सुझाता है:

- **स्थानीय AI मॉडल** (Gemma 3 4B आदि): पूरी तरह ऑफ़लाइन, मुफ़्त, कोई फ़ाइल अपलोड नहीं — गोपनीयता की पूरी सुरक्षा
- ChatGPT / Gemini जैसे क्लाउड API भी चुन सकते हैं (अपनी key चाहिए)
- व्यवस्थित करने से पहले **पूर्वावलोकन और पुष्टि**, **पूर्वाभ्यास मोड** (फ़ाइलों को छुए बिना केवल परिणाम देखें) और **पूर्ववत करें** समर्थित
- डिफ़ॉल्ट रूप से केवल एक स्तर के श्रेणी फ़ोल्डर बनाता है, फ़ोल्डर विखंडन से बचाता है

### 🧹 फ़ाइल सफ़ाई
फ़ोल्डर को स्कैन करके "शायद अनावश्यक" फ़ाइलों की चार श्रेणियाँ गिनकर दिखाता है:

| श्रेणी | विवरण |
|------|------|
| कचरा / कैश फ़ाइलें | `.tmp` `.log` `.cache` `.DS_Store` `Thumbs.db` आदि |
| डुप्लिकेट फ़ाइलें | बिल्कुल समान सामग्री वाली फ़ाइलें (हैश तुलना), अतिरिक्त प्रतियाँ चिह्नित |
| बड़ी फ़ाइलें | 100 MB से बड़ी फ़ाइलें, आकार के अनुसार क्रमबद्ध |
| खाली फ़ोल्डर / शून्य-बाइट फ़ाइलें | खाली डायरेक्टरी और 0 आकार की फ़ाइलें |

**🛡 सुरक्षा वचन: यह उपकरण केवल गिनता और स्थान बताता है — कभी भी कोई फ़ाइल हटाता या स्थानांतरित नहीं करता।** हर आइटम से एक क्लिक में उसके फ़ोल्डर तक पहुँचें, और स्वयं पुष्टि करके मैन्युअल रूप से हटाएँ।

## 📥 डाउनलोड और इंस्टॉल

### macOS (Apple Silicon)
1. [Releases](https://github.com/echoyu1025-a11y/file-butler/releases) पेज से `.dmg` डाउनलोड करें
2. उसे खोलें और ऐप को "Applications" फ़ोल्डर में खींचें
3. पहली बार खोलने पर रोका जाए तो: **ऐप पर राइट-क्लिक → Open** (ऐप Apple द्वारा नोटराइज़ नहीं है)

### Windows
- [Actions](https://github.com/echoyu1025-a11y/file-butler/actions) पेज पर सबसे हालिया सफल बिल्ड से `file-butler-windows` आर्टिफ़ैक्ट डाउनलोड करें
- या [docs/UPSTREAM_README.md](../UPSTREAM_README.md) के निर्देशों के अनुसार सोर्स से बिल्ड करें

## 🚀 उपयोग के तीन चरण

1. **मॉडल सेट करें** (केवल पहली बार): शुरू करने के बाद स्थानीय LLM चुनें (अनुशंसित, मुफ़्त और ऑफ़लाइन) और डाउनलोड करें, या ChatGPT / Gemini API key भरें
2. **व्यवस्थित करें**: बाईं ओर "फ़ाइल व्यवस्थापन" चुनें → फ़ोल्डर चुनें → "फ़ोल्डर विश्लेषण" दबाएँ → हर श्रेणी की पुष्टि करें → लागू करें
3. **सफ़ाई करें**: बाईं ओर "फ़ाइल सफ़ाई" चुनें → फ़ोल्डर चुनें → स्कैन करें → चार श्रेणियों के परिणाम देखें → क्लिक करके फ़ाइल के स्थान पर जाएँ और स्वयं निपटाएँ

इंटरफ़ेस के निचले-दाएँ कोने से कभी भी भाषा बदल सकते हैं (15 भाषाएँ); श्रेणी परिणामों की भाषा स्वतः इंटरफ़ेस भाषा का अनुसरण करती है।

## 🎨 अपस्ट्रीम की तुलना में इस संस्करण के बदलाव

- बिल्कुल नया बायाँ साइडबार लेआउट (फ़ाइल व्यवस्थापन / फ़ाइल सफ़ाई दो पेज) + हल्की सफ़ेद थीम
- बिल्कुल नया ऐप आइकन
- नई केवल-पढ़ने वाली "फ़ाइल सफ़ाई" स्कैन सुविधा (अपस्ट्रीम में नहीं है)
- इंटरफ़ेस भाषा और AI श्रेणी आउटपुट भाषा एक ही सेटिंग में मिलाई गईं, डिफ़ॉल्ट सरलीकृत चीनी
- श्रेणी विभाजन अनुकूलित: डिफ़ॉल्ट एकल-स्तरीय श्रेणी फ़ोल्डर, प्रॉम्प्ट सामान्य बड़ी श्रेणियों में विलय बाध्य करते हैं
- संपूर्ण सरलीकृत चीनी स्थानीयकरण

## 🔧 सोर्स से बिल्ड करना

macOS / Windows / Linux के पूर्ण बिल्ड चरण [docs/UPSTREAM_README.md](../UPSTREAM_README.md) में देखें। macOS त्वरित संस्करण:

```bash
brew install qt curl jsoncpp sqlite openssl fmt spdlog mediainfo cmake git pkgconfig libffi
git clone --recursive https://github.com/echoyu1025-a11y/file-butler.git
cd file-butler
cmake -S external/libzip -B external/libzip/build -DBUILD_SHARED_LIBS=OFF -DBUILD_DOC=OFF \
  -DENABLE_BZIP2=OFF -DENABLE_LZMA=OFF -DENABLE_ZSTD=OFF -DENABLE_OPENSSL=OFF \
  -DENABLE_GNUTLS=OFF -DENABLE_MBEDTLS=OFF -DENABLE_COMMONCRYPTO=OFF -DENABLE_WINDOWS_CRYPTO=OFF
cmake --build external/libzip/build
./app/scripts/build_llama_macos.sh --arm64
cd app && make -j8 && ./bin/aifilesorter
```
